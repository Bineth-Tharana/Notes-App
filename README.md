# 📝 Notes App

A simple and user-friendly Notes App built to help users create, edit, organize, and manage notes efficiently. This project demonstrates CRUD (Create, Read, Update, Delete) operations with a clean and responsive user interface.

## 🚀 Features

- ➕ Create new notes
- 📝 Edit existing notes
- 🗑️ Delete notes
- 📋 View all saved notes
- 💾 Local data storage
- 📱 Responsive design
- 🔍 Easy note management

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## ⚡ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Bineth-Tharana/Notes-App.git
```

### 2. Navigate to the Project Folder

```bash
cd Notes-App
```

### 3. Open the Application

Simply open the `index.html` file in your browser.

## 🎯 How to Use

1. Enter your note title and content.
2. Click the **Add Note** button.
3. Edit notes whenever needed.
4. Delete notes you no longer need.
5. Notes are automatically saved in local storage.
